#微信示例程序

##这是一个通过【spring websocket】模拟微信消息推送的示例程序！

#### 本程序采用的springboot框架：
* spring-boot-starter-web
* spring-boot-starter-data-rest
* spring-boot-starter-thymeleaf
* spring-boot-starter-websocket

#### 前端采用的javascript框架：
* bootstrap
* jquery
* sockjs
* stomp-websocket

##### 具体细节参见工程build.gradle
